# CutList - ჭრის ოპტიმიზაციის პროგრამა

**CutList** არის პროგრამა, რომელიც ეხმარება ხელოსნებს, ავეჯის მწარმოებლებსა და დიზაინერებს, ეფექტურად გადაანაწილონ მართკუთხა დეტალები დიდ ფილებზე (მაგ. ლამინატი, მდფ) მარტივი დასაჭრელა ფორმატით და მასალის მინიმალური დანაკარგით.

პროგრამას აქვს გრაფიკული ინტერფეისი სადაც შეგიძლიათ შეიყვანოთ მონაცემები და პროგრამა ავტომატურად დაითვლის განლაგების სხვადასხვა ვარიანტს, გაჩვენებთ მათ ვიზუალურად, რათა შეაფასოთ თქვენივე მოთხვონილებით და გაძლევთ მონაცემების ექსპორტის საშუალებას შესაბამისი ნახევრად-ავტომატური საჭრელი დაზგისთვის.

---

## ძირითადი ფუნქციები

* **მასალების მართვა:** შეიყვანეთ თქვენი ფილების ზომები მარტივ, მოსახერხებელ ცხრილში.
* **ტექსტურის კონტროლი (ბრუნვის შეზღუდვა):** მიუთითეთ, უნდა შენარჩუნდეს თუ არა დეტალის ორიენტაცია (მაგალითად, ხის ფაქტურის/ტექსტურის მიმართულების შესანარჩუნებლად).
* **ავტომატური გამოთვლა:** პარალელურ რეჟიმში უშვებს რამდენიმე ალგორითმს საუკეთესო და ყველაზე ეკონომიური ვარიანტების მოსაძებნად.
* **შედეგები:** პროგრამა ფილტრავს ყველა ალგორითმისგან მიღებულ შედეგს და გიტოვებთ მხოლოდ შეძლებისდაგვარად კარგ ვარიანტებს.
* **ნახაზი:** ნახაზებით გაჩვენებთ დეტალების განლაგებას ფილებზე და მორჩენილ ნარჩენ ნაჭრებს.
* **პროექტების ისტორია:** შეგიძლიათ შეინახოთ და ნებისმიერ დროს თავიდან ჩატვირთოთ თქვენი სამუშაო.
* **XML ექსპორტი:** გადაიყვანეთ ჭრის შედეგები XML ფორმატში, რომელიც თავსებადია შესაბამის ჭრის დანადგარებთან.

## როგორ დავაინსტალიროთ და გავუშვათ

### 1. პროექტის კლონირება
პირველ რიგში, ჩამოტვირთეთ პროექტი Git-ის საშუალებით:

```bash
git clone https://github.com/LevanMakaradze/cutlist-app.git
cd cutlist-app
```

### 2. მომზადება
დარწმუნდით, რომ თქვენს კომპიუტერში დაინსტალირებულია **Python 3.9** ან უფრო ახალი ვერსია (გატესტილია **Python 3.14**-ზე).

### 3. ბიბლიოთეკების ინსტალაცია
გახსენით ტერმინალი (Command Prompt), გადადით პროექტის საქაღალდეში და გაუშვით ბრძანება:

```
pip install -r requirements.txt
```

### 4. პროგრამის გაშვება
პროგრამის ჩასართავად გაუშვით:
```
python main.py
```

2. **დეტალების შეყვანა:** ჩანართში **"პროექტი"** ჩაწერეთ თქვენი დეტალების სიები:
   * მიუთითეთ სახელი, სიგრძე, სიგანე და რაოდენობა.
   * თუ დეტალს აქვს ხის ფაქტურა და მისი შემოტრიალება ჭრისას არ შეიძლება, მონიშნეთ სვეტი **"ტექსტურა"**.
   * მონიშნეთ გამოსაყენებელი მასალა ჩამოსაშლელი სიიდან.
3. **გამოთვლა:** დააჭირეთ ღილაკს **"ავტომატური განლაგება"**.

5. **ექსპორტი:** შეგიძლიათ შეინახოთ პროექტი ისტორიაში **"პროექტი"** ჩანართიდან ან გააკეთოთ სასურეველი ნახაზის XML ფაილის ექსპორტი გვერდიდან **"განლაგება"** შედეგების მიღების შემდეგ.
